To download the source files for LinuxCNC:
git clone https://github.com/LinuxCNC/linuxcnc.git linuxcnc
cd linuxcnc
git checkout 201288d
Follow the build instructions at <http://linuxcnc.org/docs/devel/html/code/building-linuxcnc.html>
git clone https://github.com/sheetcam/CNCRemote.git cncremote
cd cncremote
git checkout 8edf140
Install CodeBlocks <www.codeblocks.org>
Run CodeBlocks from a LinuxCNC rip-environment
Using CodeBlocks open cncremote/drivers/linuxcnc/server/linuxcncserver.cbp and build it.
For more information on building CNCRemote see cncremote/README.txt
